## lexical analysis using flex and c
This project includes a basic lexical analyzer that identifies;
keywords (int,float,char)
Identifiers
Numbers
Operators (+ - * / =)
Parentheses and semicolons
## Example input
   C
   int x=5;
   EXAMPLE OUTPUT
   keywoard;int
   Identifier;x
   Assignment Operator; =
   Number;5
   Semicolon; ;
